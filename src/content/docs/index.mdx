## Run it in one line

The same command on macOS, Linux, and Windows — no volume, no flags:

```bash
docker run -p 8080:8080 ghcr.io/omercelikdev/mockifyr
```

- **Mock surface** — `http://localhost:8080`
- **Admin API** — `http://localhost:8080/__admin`
- **Dashboard** — `http://localhost:8080/__mockifyr`

## Honest about the gaps

Mockifyr is verified by running the same scenario through both engines and diffing the result — the
reference is a real WireMock instance, not a claim. Where a behaviour is deliberately not reproduced,
it is recorded rather than glossed over: the delay distributions that are parsed, the matchers that
belong to WireMock Cloud, the helpers that do not exist despite appearing in some references.

Read [known limitations](/limitations/) before you migrate. It is the page most likely to save you an
afternoon.
